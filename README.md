# Đồ án Công nghệ phần mềm

## Scripts

### Chạy backend
#### `cd backend`
Vào thư mục backend
#### `python manage.py runserver`
Chạy backend server
### Chạy frontend
#### `cd frontend`
Vào thư mục frontend
#### `npm install`
Chạy khi mới pull về để tải node module.
#### `npm start`
Chạy server local.
### Nhập cơ sở dữ liệu
Tạo cơ sở dữ liệu tên doan và import file doan.sql

## Git Commands

#### `git --no-pager log > log.txt`
Export file log.txt
